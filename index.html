Konfigurasi Database Supabase
Cara mendapatkan URL dan Key:
1. Buka supabase.com → pilih project
2. Klik Settings (⚙️) → API
3. Copy Project URL dan anon public key
4. Paste di file index.html bagian atas
⚠️ URL dan Key dikonfigurasi langsung di file index.html sebelum deploy — bukan di sini.
